\# Payment Reconciliation \& Settlement Simulator



\## Overview

This project simulates how fintech platforms reconcile customer payments with bank settlement reports.



\## Features

\- Matches successful payments with settlements

\- Detects amount mismatches

\- Identifies missing settlements

\- Flags extra settlements



\## Tech Stack

\- Python

\- CSV files



\## How to Run

```bash

python reconcile.py
